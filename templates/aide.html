Notre application est faite pour les agent de la saa pour realiser leur stattistiques qu'ils feront avec excel pour bien l 'utiliser suiverz ces étapes suivante :

  • étape 1

    Cliquer sue le bouton inscription dans la page d'acceuille, choisissez votre département dont vous souhaiter enter, puis saisaissez vos informations sur le formulaire et cliquer sur s'inscrire pour accéder a votre espace agent spécifique a vous.

  • étape 2

    Si vous avez déja un compte, vous allez cliquer sur l'un des département que vous voulez, et un formulaire de connexion apparait, vous saisissez votre nom d'utilisateur et votre mot de passe y vous accédez directement a votre espace.

  • étape 3

    Une fois dans votre espace, vous allez vous dériger vers le menu et cliquer sur le bouton service, pour choisir le service dont vous vous travaillez et cliquer dessus. dans chaque service vous truvez les états statistique à réaliser.

  • étape 4

    Vous êtes donc dans votre service, donc pour avoir l'état que vous voulez, vous allez choisir l'état correspondant et cliquez dessus, un formulaire s'affichera dont vous allez saisir les données d'entrés qui spécifie chaque etat avec les données qu'on veut.
